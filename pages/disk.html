About Disk Scheduling

Disk scheduling algorithms determine the order in which disk I/O requests are serviced to minimize seek time and improve performance.

FCFS: First Come First Serve
SSTF: Shortest Seek Time First
SCAN: Elevator Algorithm
C-SCAN: Circular SCAN
