# OS Algorithm Simulator

A comprehensive web-based visualization tool for Operating System algorithms. This project helps users understand core OS concepts through interactive simulations, charts, and step-by-step visual outputs.

## Features

- Process Scheduling Algorithms
  - First Come First Serve (FCFS)
  - Shortest Job First (SJF)
  - Shortest Remaining Time First (SRTF)
  - Round Robin
  - Priority Scheduling

- Memory Management Algorithms
  - First Fit
  - Best Fit
  - Worst Fit
  - FIFO Page Replacement
  - LRU Page Replacement
  - Optimal Page Replacement

- Deadlock Avoidance
  - Banker's Algorithm

- Disk Scheduling Algorithms
  - FCFS
  - SSTF
  - SCAN
  - C-SCAN

## Visualizations

- Gantt charts for process scheduling
- Memory block diagrams for memory allocation
- Page replacement tables
- Disk head movement graphs
- Step-by-step output for better understanding
